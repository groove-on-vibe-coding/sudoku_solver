======================================
数独（すうどく）ソルバー for Windows
======================================

【概要】
このアプリケーションは、数独パズルを解いたり、新しいパズルを生成したりするためのツールです。
数独は 9x9 のグリッドに 1 から 9 までの数字を入れる論理パズルで、各行、各列、各 3x3 ブロックに
同じ数字が重複しないようにするルールがあります。

【使い方】
1. 起動方法
   sudoku_solver.exe をダブルクリックして起動してください。

2. メイン機能
   - 「読み込み」: 保存済みの数独ファイル(.txt)を開きます
   - 「生成」: 新しい数独パズルを生成します（難易度を選択できます）
   - 「解く」: 現在の盤面を自動的に解きます
   - 「リセット」: 盤面を初期状態に戻します
   - 「保存」: 現在の盤面を.txtファイルとして保存します
   - 「終了」: アプリケーションを終了します

3. 数字の入力
   - 空のマスをクリックして選択し、キーボードで 1 〜 9 の数字を入力
   - 数字を消すには、0 または Delete キーを押します
   - 矢印キーで選択マスを移動できます

4. 盤面の解法
   - 「解く」ボタンをクリックすると自動的に解法が始まります
   - 解法中は「解く」ボタンが「中断」ボタンに変わります
   - 解法速度はスライダーで調整できます（左が遅く、右が速く）

5. サンプルファイル
   同梱の sample_easy.txt、sample_medium.txt、sample_hard.txt を
   「読み込み」ボタンから開くことで、サンプル問題を試すことができます。

6. 言語設定
   - 言語は日本語と英語に対応しています
   - 言語を変更するには以下の2つの方法があります：
     a) ui_setting.json ファイルをテキストエディタで開き、"language" の値を "ja"（日本語）または "en"（英語）に変更
     b) コマンドプロンプトまたはPowerShellから以下のように起動する：
        コマンドプロンプト: start "" "sudoku_solver.exe" --language en
        PowerShell: Start-Process -FilePath "sudoku_solver.exe" -ArgumentList "--language en"

【注意事項】
- ルール違反のある盤面（同じ行/列/ブロックに同じ数字）は解けません
- 解けない問題の場合はその旨のメッセージが表示されます
- 大きな問題の場合、解法に時間がかかることがあります

【問題が発生した場合】
アプリケーションが起動しない場合は、以下をご確認ください：
1. Windows Defender などのセキュリティソフトがブロックしていないか
2. 実行に必要な権限があるか
3. PC のスペックが最低要件を満たしているか
